## Overview
Fake news detection is a critical problem in modern information systems. This project goes beyond raw accuracy by analyzing how well different machine learning models generalize to unseen data. Techniques such as regularization, pruning, feature selection, and k-fold cross-validation are applied to reduce overfitting and improve reliability.

## Models Used
- Logistic Regression  
- Decision Tree  
- Random Forest  
- Gradient Boosting  

## Methodology
The workflow includes data preprocessing and feature extraction, model training using multiple algorithms, application of overfitting reduction techniques, k-fold cross-validation, and a comparative evaluation of model performance.

## Evaluation Focus
Model performance is assessed using accuracy, cross-validation scores, and training vs testing performance comparisons to better understand generalization behavior.

## Key Insights
Ensemble methods such as Random Forest and Gradient Boosting showed stronger generalization capabilities, while regularization and pruning significantly helped reduce overfitting across models.

## Disclaimer
This project is intended for academic and research purposes. Model predictions should not be treated as definitive judgments of news authenticity.
